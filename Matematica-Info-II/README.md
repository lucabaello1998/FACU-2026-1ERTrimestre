# Matemática para Informática II (793) - 2026 C1

## Información de la materia

- **Modalidad:** Mixta (presencial / virtual sincrónica)
- **Parciales:** 2 obligatorios + recuperatorios
- **Trabajos Prácticos:** 2 (TPo)
- **Final:** Integrador (preparación desde 01/08)

## Temas principales

1. **Principios básicos de conteo y análisis combinatorio**
   - Repaso algebraico
   - Conteo simple
   - Combinatoria con repetición

2. **Relaciones**
   - Producto cartesiano
   - Aplicación práctica
   - Manejo matricial de operaciones
   - Composición de relaciones

3. **Relaciones binarias**
   - Propiedades (reflexividad, simetría, transitividad, etc.)
   - Clasificación

4. **Relaciones de equivalencia**
   - Clases de equivalencia
   - Conjunto cociente
   - Teorema fundamental de relaciones de equivalencia
   - Congruencia módulo n

5. **Relaciones de orden**
   - Conjuntos ordenados
   - Diagrama de Hasse
   - Elementos comparables
   - Elementos notables (máximo, mínimo, maximal, minimal)

6. **Principio de inducción completa**
