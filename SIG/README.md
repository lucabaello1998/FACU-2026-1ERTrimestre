# Sistemas de Información Geográfica (SIG) - 2026

## Información de la materia

- **Modalidad:** Mixta (virtual / presencial)
- **Teórico:** Virtual semanal
- **Práctico:** Presencial (desde clase 4)
- **Software principal:** QGIS + R + RStudio

## Instalaciones necesarias

- [QGIS](https://www.qgis.org/es/site/) — mapas, capas, visualización
- [R](https://cran.r-project.org/) + [RStudio](https://posit.co/download/rstudio-desktop/) — análisis espacial y teledetección

## Programa de la materia

| Bloque | Tema |
|--------|------|
| 01 | Introducción a GIS, objetivos, tecnologías |
| 02 | Posicionamiento: coordenadas, sistemas de referencia, proyecciones, datums |
| 03 | Modelos de datos: vectorial, raster, SHP, GeoTIFF, KML |
| 04 | Fuentes de datos espaciales |
| 05 | Operaciones sobre datos |
| 06 | Creación de mapas en QGIS |
| 07 | Teledetección: sensores remotos, imágenes satelitales |
| 08 | Bases de datos espaciales: tipos, consultas, índices |
| 09 | Servidores de mapas: WMS, WFS |
| 10 | Clientes de mapas: desktop y web |
